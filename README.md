# 📡 OTT Growth Copilot: AI-Powered Analytics MVP

### 🚀 Overview
Este repositorio contiene un **Copiloto de Growth para plataformas OTT**, diseñado para transformar métricas crudas en decisiones estratégicas. Utiliza IA Generativa para actuar como un "Growth Manager Virtual", detectando anomalías en KPIs críticos y sugiriendo planes de acción inmediatos.

### 🧠 Core Functionalities
* **Metric Anomaly Detection:** Identificación estadística de desviaciones en Churn Rate, CAC y ARPU.
* **AI Strategic Reasoning:** Agente basado en LLM que analiza el contexto de negocio y recomienda estrategias de retención y LTV.
* **Interactive Dashboards:** Visualización dinámica de la salud de la plataforma utilizando Plotly.

### 🛠 Technical Stack
* **LLM Engine:** OpenAI GPT-4o-mini
* **Orchestration:** LangChain
* **Analysis:** Pandas, NumPy
* **Visualization:** Plotly
* **Environment:** Python 3.10+
